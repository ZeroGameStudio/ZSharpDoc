构建工具

Z# 有一些编辑器工具可以帮助开发者简化开发流程。

控制台命令​

z#.gens (Generate Solution)​

生成解决方案，当你修改了 Z# 项目文件后，需要执行这条命令刷新解决方案。

z#.geng (Generate Glue)​

生成胶水代码，当你修改了 C++ 代码导致虚幻类型系统的元数据发生变化后，需要执行这条命令同步到胶水代码。

你可以通过 assemblies=name1,name2... 参数指定要生成胶水代码的程序集，这样可以避免每次都全量生成，节省时间。

注意

跳过此步可能导致脚本调用引擎时产生未定义行为！
