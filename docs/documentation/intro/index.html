简介

Z# 是一款基于虚幻引擎的脚本解决方案。它在虚幻引擎中集成了 .NET 运行时，使你可以用 .NET 技术栈在虚幻引擎上进行开发。

Z# 的优势​

  • 使用 .NET 技术栈，兼顾了开发效率与运行效率。
  • .NET 同时拥有编译期的强类型系统和完备的运行时反射机制，兼顾了程序的可维护性和灵活性。
  • .NET 拥有强大的生态圈，可以使用大量第三方组件帮助开发。
  • C# 优雅的语法和强大的特性带来最佳的开发体验。
  • 自动生成带有静态类型信息的高性能胶水代码，与虚幻引擎无缝交互。

Z# 的特色​

  • 与虚幻引擎运行时和编辑器高度集成，获得接近原生开发的体验。
  • 提供内置的适用于虚幻引擎的异步模型和编程接口，集成了 C# 的异步语法，大大降低了异步编程的心智负担。
  • 提供适用于 Z# 的全套静态分析工具，让错误尽早暴露出来。
注意

Z# 最主要的开发目的是自用，开源更多是为了学习和交流。 因此，Z# 在泛用性、版本兼容性、生态建设等方面与其他开源项目相比会略有不足。 同时，Z# 的架构设计中普遍存在着作者的主观审美，不一定符合所有人的习惯。

如果你正在为你的商业项目寻找一款脚本引擎，建议选择市面上更通用的解决方案。
