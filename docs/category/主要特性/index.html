主要特性

📄️ 互操作

Z# 作为一个脚本引擎，需要提供的核心能力就是互操作。

📄️ Emit

Z# 提供了一种将托管程序集元数据转换为虚幻类型系统元数据（反射信息）的机制，这个机制被称为 Emit。

📄️ 引擎接口

Z# 内置导出了一些常用的引擎 API。
